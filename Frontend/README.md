# Frontend

React + TypeScript + Vite app.

## Run

PowerShell may block `npm` (because `npm.ps1` is a script). Use the `cmd` shim:

```bat
cd Frontend
npm.cmd install
npm.cmd run dev
```

Then open `http://localhost:5173`.

## Commands

```bat
npm.cmd run build
npm.cmd run preview
npm.cmd run lint
```
